Option / 參數

Baisc Example / 基本範例 
	$.jkrolling.apply( $('#demo'), './demo.jpg');

第二個參數為圖片，目前是一張很長的圖(未來可以多圖串接)

特別設定 
  $.jkrolling.apply( $('#demo'), './demo.jpg', {
      sliceNum : 300, // 切片數量 | set slice num (for more smooth)
      centerKeepRange : 0.2, // 中間保留範圍
      rollingDeep : 0.5, // 滾動深度
      rollingType : $.jkrolling.type_roller // 滾動類型
  });
第三個參數為客制化設定參數，說明於下方

參數說明

sliceNum : 切片數量，影響順暢度和鋸齒

centerKeepRange : 中間保留範圍，越大中間越不會滾動
